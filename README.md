## 🚀 Featured Projects

### 🧠 RAG-Based Resume Analyzer
> A Retrieval-Augmented Generation system that analyzes resumes and answers queries contextually.

- Designed document ingestion, chunking, vector embedding, and similarity search using **FAISS**
- Integrated an **LLM** (LLaMA / OpenAI) with retrieved resume context to generate accurate, grounded responses — reducing hallucinations significantly
- Built a clean **FastAPI** backend to expose the pipeline as a REST API

`Python` `FastAPI` `LangChain` `FAISS` `LLaMA` `MySQL` `Git`

- Performed **EDA and feature engineering** on variables like population, income, parking cost, and pricing
- Implemented **regression-based ML models** and evaluated with appropriate metrics
- Designed the system to support business decisions like pricing strategy and demand planning

`Python` `Pandas` `NumPy` `Scikit-Learn` `MySQL` `Matplotlib`
